What is CyberSafety? Cyber security focuses on protecting computer systems from unauthorised access or being otherwise damaged or made inaccessible.

How does CyberSafety work? CyberSafety requires designing secure application architectures, writing secure code, implementing strong data input validation, threat modeling, etc. to minimize the likelihood of any unauthorized access or modification of application resources.

Types of Cyber Attacks

1: Social Engineer : Social engineering refers to all techniques aimed at talking a target into revealing specific information or performing a specific action for illegitimate reasons.

2: Password Attacks: Password attacks are malicious ways hackers attempt to gain access to your account.

2: MITM: A man in the middle (MITM) attack is a general term for when a perpetrator positions himself in a conversation between a user and an application

4: Ransomware: Ransomware is a type of virus which prevents you from accessing your device and the data stored on it, usually by encrypting your files

5: Phishing: Phishing is when attackers send emails designed to trick people into falling for a scam.

6: Botnets: A botnet is defined as a cyberattack that uses multiple networked devices to run one or more bots on each device and then uses this swarm of infected devices to attack a server, company website, or other devices or individuals.

Basic principles we should follow to ensure Cyber Safety

1: Use antivirus software. Like: Avast, Bitdefender, Norton.

3: Use strong passwords, and ideally a password manager to generate and store unique passwords. One strong Password Generator is 1password.com

2: Think before you click.

Things we can do to improve Cyber Security

To Improve Cyber safety, we can update our software. Updating software and system security updates quickly limits possible exposure to vulnerabilities.
